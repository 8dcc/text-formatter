# text-formatter
**Formats words to the N column of a text**

### Building
Simply run:
```console
$ git clone https://github.com/r4v10l1/text-formatter

$ cd text-formatter

$ make
...

$ ./text-formatter -h
...
```

### Installing
For adding the program to the path, run:
```console
$ sudo make install
...
```
